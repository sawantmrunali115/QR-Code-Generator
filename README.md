<h1 align="left">QR Code Generator</h1>

###

<p align="left">The QR Code Generator is a simple web application that allows users to instantly generate QR codes for any text or URL. Whether you want to share a website link, contact details, or short message, this tool quickly converts your input into a scannable QR code image.<br><br>Features:<br>-  Fast & Lightweight – Uses a public QR Code Server API to generate QR codes without heavy processing.<br>- Instant QR Code Creation – Generate a QR code instantly for any text or URL you enter.<br>- Responsive Design – Fully optimized layout built with Bootstrap, ensuring smooth performance across all devices.<br>- User-Friendly Interface – Clean and minimal design focused on simplicity and usability.<br><br>Tech Stack:<br>Frontend:  HTML, CSS, Bootstrap, JavaScript, QR-Server API</p>
